Pexels Photo Search App

02
Pexels Photo
Search App
03
December 2022
AJAX & API Development
Try it here!

This Photo Search App is powered by Pexels API. The Pexels API is free to the public. This project turns your words into pictures that are from Pexels API. Simply enter text into the input field and see images that match the words. Each photo has a hover feature that displays the photo title, photographer name, and connects you with the photographers Pexels account to follow them. Excited about this powerful yet simple app? Connect with me.


Features:

  1. Minimalist & User Friendly Design
  2. Search Bar Displays Photos from Words
  3. Each Photo is Powered by Pexels API
  4. Hover Feature Displays Photo & Photographer Information
  5. Responsive Web Design - Compatible on all Devices

Development Tools:

  • HTML & CSS
  • JavaScript
  • Pexels API
  • AJAX
  • XMLHttpRequest
  • JSON
