Shopify Development
Dawn Theme

Evergreen Brew Store is an online aroma coffee store. The Dawn template is a great for anyone starting out in Shopify for the first time it's free! And for those who want to update their current store with some really cool features without too much hassle. There are many features like: gallery, promotional banner, image banner, and email sign up.


This store has a testimonial slider that is an additional customizable feature. Each card has name, picture, and review is user interactive by scrolling or clicking the navigation dots. Thanks to Slick, a free library for a quick startup to the testimonial slider. Work with me now!.

Dawn Theme Features:

  1. Sticky Header
  2. Customizable Sections
  3. Mega Menu
  4. Image Gallery & Slideshow
  5. Product Filters

Development Tools:

  • Shopify Template
  • HTML & CSS
  • JavaScript
  • Shopify Liquid
  • Slick Library
