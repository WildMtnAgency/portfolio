How do I add a slide to the home page?
Note: Every project has its own unique number. This is because, this way, we can filter the slide we want from the CMS collection in the home page slider. To add a new slide, it's important to give you new project a unique number. Eg: previous slide number was '03', then the new number for the new project is '04', etc. Slider CMS projects start counting from '02' because the first slide '01' is not a CMS collection.
How do I add a 'Change slides on scroll' functionality to my project's home page?
For this, you need to go to this project and go to the 'page settings' of the home page. Then, scroll down to the 'custom code' section and copy-paste everything from that project to the 'custom code' section of your project. Publish and the code should now add the 'change slide on scroll' functionality. Note: the code may not work in the designer, but that's normal. It does work on the published site.
